# Reminder App Repository

Home Assistant Add-on Repository für die Reminder App.

## Installation in Home Assistant

[![Add Repository](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FTunerDav%2Freminder-app)

Oder manuell:

1. Gehe zu **Einstellungen** → **Add-ons** → **Add-on Store**
2. Klicke auf **⋮** → **Repositories**
3. Füge hinzu: `https://github.com/TunerDav/reminder-app`
4. Suche nach "Reminder App" und installiere sie

## Add-ons in diesem Repository

### Reminder App

Family relationship manager with reminders, events and contacts for Home Assistant.

**Features:**
- Kontakt- und Familienverwaltung
- Beziehungs-Tracking
- Automatische Erinnerungen (Geburtstage, Jubiläen, Anrufe, Besuche)
- Event-Management mit Templates
- Einladungsgruppen
- Kalenderansicht

**Detaillierte Dokumentation:** [INSTALLATION.md](INSTALLATION.md)
