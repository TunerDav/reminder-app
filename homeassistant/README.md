## About

This repository contains Home Assistant add-ons for the Reminder App - a family relationship manager with reminders, events, and contacts.

Family relationship manager for Home Assistant with contacts, reminders, and event tracking.

## Installation

1. Click the following button to add this repository to your Home Assistant instance:

   [![Add Repository][add-repo-shield]][add-repo]

2. Or manually add the repository:
   - Navigate to Supervisor → Add-on Store
   - Click the menu (⋮) → Repositories
   - Add this URL: `https://github.com/TunerDav/reminder-app`

3. Find "Reminder App" in the add-on store and click Install
